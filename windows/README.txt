Persona for Windows (standalone)
================================

1. Unzip this ENTIRE folder (all files must stay together).
2. Double-click Persona.exe
3. Built-in AI loads automatically — first start may take ~30 seconds.

WHAT'S INCLUDED (~1.3 GB total)
- Built-in offline AI (no Ollama install needed)
- llama-server + Fast (~350 MB) + Balanced (~700 MB) models bundled
- Quality model (Qwen2.5 3B, ~2 GB) — download from Settings for file & memory tools

OPTIONAL:
- Run "Create Desktop Shortcut.bat" to pin Persona to your desktop.
- Or use "Run Persona.bat" if double-click does not work.

IMPORTANT:
- Do NOT move Persona.exe out of this folder.
- The "_internal" folder must stay next to Persona.exe.
- Recommended: 8 GB+ RAM (16 GB ideal for Quality model).

SETTINGS (inside the app)
- Built-in AI — default, works offline
- Model manager — Fast / Balanced / Quality
- CPU threads & GPU layers — tune performance
- Ollama / Cloud API — optional upgrades

If nothing happens when you double-click Persona.exe:
1. Use "Run Persona.bat" instead — it shows error messages and logs.
2. Check: %USERPROFILE%\.persona\startup.log
3. Check: %USERPROFILE%\.persona\error.log
4. Confirm the "_internal" folder is next to Persona.exe (unzip the whole zip).
5. Allow through Windows Defender / antivirus if prompted
