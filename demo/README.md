# Persona demo

## What is this?

**`DEMO.md`** is a *recording script* for making a 90-second screen capture video (Loom, OBS, etc.).

**`demo/index.html`** is an **interactive demo** anyone can play in the browser — no install, no AI keys. It auto-walks through:

1. Picking the **Launch a side project** template  
2. **Project mode** — Captain plans, Byte and Nova respond  
3. **Kanban board** — tasks appear and move  
4. Download links for the real app  

## Try it

### Watch the demo video (~36 seconds)

**[demo/Persona-demo.mp4](Persona-demo.mp4)** — auto-recorded walkthrough (templates → Project → board → download)

### Online (after GitHub Pages deploy)

https://armadilloarmada.github.io/personal-LLM/

### Locally

```bash
# Option A — open the file
open demo/index.html   # macOS
xdg-open demo/index.html   # Linux

# Option B — via running Persona
persona app
# Then visit http://127.0.0.1:8765/demo
```

Click **▶ Play demo** and watch (~60 seconds).

## Record a video demo

Use `DEMO.md` as your voiceover script while screen-recording the **real app** (Windows `.exe` or `persona app`). The interactive HTML demo is for sharing on the web; the video should show the actual product.
