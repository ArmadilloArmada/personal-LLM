# Download Persona for Windows

## Direct download

**Latest Windows build:**

https://github.com/ArmadilloArmada/personal-LLM/releases/latest/download/Persona-Windows-portable.zip

1. Download and unzip
2. Double-click **`Persona.exe`**
3. Done — no install, no Python, no Ollama required (demo mode works instantly)

## All releases

https://github.com/ArmadilloArmada/personal-LLM/releases

## Notes

- Windows may show "Unknown publisher" — click **More info → Run anyway**
- Your data is stored in `%USERPROFILE%\.persona`
- Optional: install [Ollama](https://ollama.com) for full local AI — Persona auto-detects it
