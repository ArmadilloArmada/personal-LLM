# DEPRECATED — use unified Persona app instead

The browser bridge (`big-brain-bridge.js`) and `install-capture.ps1` are **deprecated**.

Capture and RAG now run **server-side** in Persona (`persona-app/src/persona/web/server.py`) with Big Brain as a child API process.

See [Persona README](../README.md) and [DUAL_RAG.md](../docs/DUAL_RAG.md).
