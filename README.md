# This repo has moved

**Persona** now lives at **[github.com/ArmadilloArmada/Persona](https://github.com/ArmadilloArmada/Persona)**.

`personal-LLM` was the original repo name. The app was rebranded to Persona, then merged with **Big Brain** into the unified [`ArmadilloArmada/Persona`](https://github.com/ArmadilloArmada/Persona) monorepo. This repository is kept only so old links keep working.
