# Persona + Big Brain

**Persona** is a standalone Windows AI agent app with an integrated **Big Brain** knowledge vault. Download, unzip, double-click — no Python or Node setup for end users.

## Download (Windows)

**https://github.com/ArmadilloArmada/CursorProjects/releases/latest/download/Persona-Windows-portable.zip**

1. Download and unzip the **entire folder**
2. Double-click **Persona.exe** (or **Run Persona.bat**)
3. Use **Chat** and **Big Brain** tabs in the header

Optional: **Create Desktop Shortcut.bat**

See [DOWNLOAD.md](DOWNLOAD.md) for troubleshooting.

## What you get

- **Chat** — Solo, Group, Project, Board with built-in agents
- **Big Brain** — Vault, graph, workflows, auto chat capture
- **Demo mode** — instant, no API keys
- **Ollama** — auto-detected local AI
- **Team workspaces**, knowledge base, custom agents

## Data location

`%USERPROFILE%\.persona\` — config, vault, database, logs (local only).
